# ATM Simulator (Java)

## Overview
A console-based ATM simulation system developed using Java and OOP principles.

## Features
- PIN authentication system
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven user interface

## Tech Stack
- Java
- Object-Oriented Programming (OOP)

## Key Learnings
- Classes and objects
- Encapsulation
- Input handling using Scanner
- Control flow and logic building
